Auto-Learned STT Corrections
Every time the LLM fixes a garbled question, the correction is saved here and applied automatically next time — no manual work needed.
0
How auto-learning works:
1. STT transcribes interviewer audio (may mishear tech terms)
2. If the question misses the DB, LLM corrects the phrasing (e.g. "gill" → "GIL")
3. The correction is saved here and applied to every future STT output — silently, instantly
4. Corrections hot-reload every 5 min without restarting the server
